# @starter-kit/vue2-project

Vue 2.7.x 单体项目开发模板

## 目录结构

```sh
.
├── src
│   ├── main.ts                           入口文件
│   ├── app                               主应用组件
│   ├── assets                            静态资源
│   │   ├── fonts
│   │   ├── images
│   │   └── styles
│   ├── configs                           应用配置
│   ├── composables                       组合式API
│   ├── components                        基础组件
│   ├── features                          功能组件
│   ├── layouts                           布局组件
│   ├── pages                             页面组件
│   │   └── home
│   │       ├── index.tsx
│   │       ├── page.config.ts
│   │       ├── routes.config.ts
│   │       └── styles.module.less
│   ├── plugins                           插件
│   ├── router                            路由
│   ├── services                          服务、API
│   ├── stores                            全局状态 Store
│   ├── types                             类型扩展
│   └── utils                             工具函数
├── docs                                  文档
├── e2e                                   端到端测试
├── public                                静态资源
├── vite.config.ts                        Vite 配置
├── vitest.config.ts                      Vitest 配置
└── README.md
```
